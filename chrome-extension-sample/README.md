# Chrome拡張機能サンプル

シンプルなChrome拡張機能のサンプルです。現在のタブ情報を表示し、ページの背景色を変更したりアラートを表示する機能があります。

## 機能

- 現在のタブのURLとタイトルを表示
- ページの背景色をランダムに変更
- アラートの表示

## インストール方法

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnterキーを押す
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
5. このフォルダ（`chrome-extension-sample`）を選択
6. 拡張機能が読み込まれ、ツールバーにアイコンが表示される

## 使用方法

### 基本操作
1. Chromeのツールバーにある拡張機能のアイコンをクリック
2. ポップアップウィンドウが表示される
3. 現在のタブのURLとタイトルが自動的に表示される

### 背景色変更機能
1. 任意のWebページを開く
2. 拡張機能のポップアップを開く
3. 「背景色を変更」ボタンをクリック
4. ページの背景色がランダムな色に変わる

### アラート機能
1. ポップアップで「アラート表示」ボタンをクリック
2. 「Hello from Chrome Extension!」というアラートが表示される

## 動作確認

1. **拡張機能の表示確認**
   - ツールバーにアイコンが表示されているか確認
   - アイコンが見えない場合は、パズルピースアイコン（拡張機能メニュー）をクリックして探す

2. **ポップアップの表示**
   - アイコンをクリックしてポップアップが開くか確認
   - タブ情報が正しく表示されるか確認

3. **機能テスト**
   - 背景色変更ボタンが動作するか確認
   - アラートボタンが動作するか確認

## トラブルシューティング

### 何も起きない場合
- `chrome://extensions/` で拡張機能を一度無効にして再度有効にする
- ページを再読み込みしてから試す
- 開発者ツール（F12）のコンソールでエラーを確認

### 権限エラーが表示される場合
- manifest.jsonに必要な権限（`activeTab`, `scripting`）が含まれているか確認
- 拡張機能を再読み込みする

## ファイル構成

```
chrome-extension-sample/
├── manifest.json    # 拡張機能の設定ファイル
├── popup.html      # ポップアップのHTML
├── popup.js        # ポップアップの機能実装
├── popup.css       # ポップアップのスタイル
└── README.md       # このファイル
```

## 技術仕様

- **Manifest Version**: 3
- **必要な権限**: `activeTab`, `scripting`
- **対応ブラウザ**: Google Chrome（最新版推奨）
